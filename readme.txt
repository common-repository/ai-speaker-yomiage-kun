This plugin can set the gutenberg block which speech atricle text.

== Description ==

This plugin makes gutenberg block which is "text speech block".

* This block can only be selected once per article.
* You can freely set the text at the top of the text-to-speech block.

You can edit "speech blocks" as you like!
The settings options are following.
* Button color
* Speech settings
    * pitch
    * rate
    * value


In addition, regular updates ensure that the latest plugin is always available.


== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “AI speaker ~YOMIAGE-KUN~”.
3. Select the “AI speaker ~YOMIAGE-KUN~” plugin and click “Install”.
4. Activate the plugin.

OR...

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
